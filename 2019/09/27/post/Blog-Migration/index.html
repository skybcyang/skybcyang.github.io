搬家&&博客迁移&&立flag时间

2019.09.27 SkybcYang  热度 ℃

工作一年，租房到期搬家，上家房东把房间给卖了，不能续租了。希望早日有自己的小窝～

刚毕业时和大学同学一起住的，三个人挤两室一厅，这次看房的时候，我们找到一个复式的，一层是两室一厅，二层是一个阁楼。

我看中了阁楼的清净，在能力范围内对阁楼进行了小幅度改造和装饰。

插播一下：博客回忆录

大学的时候也对博客很感兴趣，当时搭建了一个wordpress博客，用于记录考研的日志，同时也给一个朋友开了一个号用来交流。不过后来考研自闭了，就没有继续写了，租的服务器到期了，导出的备份存在硬盘上在后来的事故中也弄丢了，非常可惜。

后来就转向hexo + github pages搭建博客，这次对博客也进行了搬迁，主要还是名字的问题吧，想要好好维护一个github的账号，我常用的skybc名字已被占用，之前使用的kinosas源于《奇诺之旅》，是我在实习公司的花名，想了想还是申请了现在的名字skybcyang。

博客搬迁时趁着换主题美化的空档，抽空对内容重新审视进行小幅度修订。

万事俱备了吧，就差趁着豪情立几个在一年后房子到期之前看起来应该能达到的flag

  • 看20本技术相关的书籍
  • 博客数目超过50篇
  • 做一些感兴趣的开源项目
  • 多思考，多动手，多输出
  • 做好断舍离的修炼
  • 坚持锻炼身体
  • 通关三款3A大作
  • 重玩饥荒、缺氧、starbound
  • 拍照or画画入门
  • 加强英语和日语学习

我们都是普通人，每天吃饭睡觉尽心尽力地去完成我们的工作，日子一天天流淌过去，刚刚结束完一件“重要”的事还没来得及释然，又慌慌张张地去准备下一件“重要”的事情，人可真渺小呢。不过希望每个人都能得暂时的安乐和永久的解脱。
